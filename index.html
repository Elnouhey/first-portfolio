clean code

Clean code is code that is easy to understand and easy to change. Some say "Clean code always looks like it was written by someone who cares". Others say "Beautiful code, like beautiful prose, is the result of many small decisions. The right method length here, the right object name there". My goal is to make simple, easy to read, easy to understand and easier to change it.
